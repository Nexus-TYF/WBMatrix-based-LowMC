# WBMatrix based LowMC

An implementation of [LowMC block cipher](https://eprint.iacr.org/2016/687).

## Build

```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

## Run

```
$ ./LMC
```

## Included libraries
1. [WBMatrix](https://github.com/Nexus-TYF/WBMatrix)<br>
2. [LowMC](https://github.com/LowMC/lowmc)<br>
